## Introduction
CentrePro is a trading platform designed for both novice and professional traders. It offers a comprehensive suite of tools for market analysis, strategy development, and trade execution.

## Features
- Real-time market data
- Advanced charting tools
- Customizable trading strategies
- Risk management features
- User-friendly interface

## Installation
To install CentrePro, follow these steps:
1. Download the latest version from the [official website](#).
2. Extract the downloaded files.
3. Run `setup.exe` and follow the on-screen instructions.

## Usage
To get started using CentrePro, launch the application and create an account. You can then configure your settings and begin trading.
